# Surface

Fresh start project with deployment scripts preserved from v0.

## Deployment

The project includes Docker configurations for both Node.js and Bun runtimes:

- `Dockerfile.node` - Node.js runtime container
- `Dockerfile.bun` - Bun runtime container  
- `docker-compose.yaml` - Multi-service composition for development and production

Build and deploy containers to Cloud Run using the provided Docker configurations.
